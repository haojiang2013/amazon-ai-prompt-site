AI Prompts for Amazon Listing Optimization

Maximize your A9 search algorithm visibility. Use these tested ChatGPT prompts to generate keyword-rich titles and 5-bullet point features.

Title Generation Prompt

"Act as an expert Amazon SEO copywriter. Generate 5 highly optimized Amazon product titles for [Product Name]. Include the top keywords: [Keyword 1], [Keyword 2]. Keep the length between 150-200 characters and place the main benefit at the beginning."

Bullet Points Prompt

"Write 5 Amazon bullet points for [Product Name]. Format: [ALL CAPS BENEFIT] - [Feature detail]. Ensure emotional triggers are used, and natural integration of long-tail keywords like [Keyword 3] and [Keyword 4]."

AI Prompts for Product Descriptions

Convert browsers into buyers with compelling HTML product descriptions and A+ content outlines using advanced AI instructions.

HTML Description Prompt

"Write a 2,000-character Amazon product description for [Product]. Use basic HTML tags like <b>, <br>, and <ul> for formatting. Focus on a storytelling approach that addresses the customer's pain point regarding [Pain Point]."

A+ Content Module Prompt

"Draft the text for an Amazon A+ Content 'Standard 3 Image & Text' module. Provide a strong headline and 50 words of persuasive copy for each of the following 3 features: [Feature 1], [Feature 2], [Feature 3]."
